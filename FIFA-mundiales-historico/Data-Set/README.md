# Datasets
# partidos.csv
Contiene información histórica de los partidos disputados en los Mundiales, incluyendo:
Selección local y visitante
Capitanes
Cantidad de asistentes
Fase del torneo
Fecha
Resultado
País anfitrión

# ranking_fifa.csv
Contiene información de las selecciones nacionales:
Nombre del equipo
Código internacional
Confederación
Posición en el ranking FIFA
