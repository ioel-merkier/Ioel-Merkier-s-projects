# Análisis histórico de los Mundiales de Fútbol
# Descripción
Proyecto desarrollado en Python para analizar información histórica de los Mundiales de Fútbol a partir de dos archivos CSV: una base de partidos disputados y una tabla con el ranking y la confederación de las selecciones.
El programa procesa los datos sin utilizar bibliotecas externas de análisis, aplicando funciones propias para leer archivos, filtrar registros, ordenar información, interpretar resultados y generar reportes.

# Objetivos
- Calcular el promedio de asistentes por edición del Mundial.
- Identificar resultados inesperados según el ranking FIFA.
- Analizar victorias de selecciones americanas frente a equipos europeos en fases eliminatorias.
- Reconocer a los capitanes con mayor cantidad de partidos disputados.
- Generar reportes automáticos a partir de archivos CSV.

# Herramientas utilizadas
- Python
- Módulo csv
- Manejo de archivos
- Funciones
- Listas y ciclos
- Validación de datos
- Algoritmos de ordenamiento
- Procesamiento de texto
