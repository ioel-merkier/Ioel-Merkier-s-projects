
# Salud mental: ¿Qué influye realmente?

# Descripción
Este proyecto de **Data Storytelling** analiza cómo distintos hábitos y características personales se relacionan con la salud mental. A partir de un conjunto de datos de **300 personas**, se exploran variables como las horas de sueño, la actividad física, la edad, el tiempo frente a pantallas y la interacción social para identificar los factores con mayor impacto sobre el bienestar.
Los resultados se presentan mediante una historia visual e interactiva desarrollada en **Flourish**, permitiendo transformar los datos en insights claros y fáciles de interpretar para cualquier audiencia.

# Objetivos
- Analizar los principales factores asociados al bienestar mental.
- Identificar patrones y relaciones entre hábitos cotidianos y salud mental.
- Comunicar hallazgos mediante visualizaciones interactivas.
- Aplicar técnicas de **Data Storytelling** para convertir datos en una narrativa comprensible.

# Herramientas utilizadas

- **Microsoft Excel** – Limpieza y preparación de datos.
- **Flourish** – Desarrollo de visualizaciones interactivas.
- **Data Storytelling** – Construcción de la narrativa y comunicación de insights.

# Principales variables analizadas
- Horas de sueño
- Edad
- Nivel de actividad física
- Duración del ejercicio
- Tipo de ejercicio
- Tiempo de pantalla
- Alimentación
- Interacción social
- Estrés
- Ansiedad
- Depresión
- Nivel de felicidad
- Puntaje de salud mental

# Resultado
El proyecto demuestra cómo una adecuada combinación de análisis de datos y visualización interactiva permite descubrir patrones relevantes sobre los factores que influyen en la salud mental y comunicar los resultados de forma clara, intuitiva y basada en evidencia.
